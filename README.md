### Prerequisites
Make sure you have:
- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed

- Device setup for your target platform(s)  

### Installation
1. **Clone the repo**
   ```bash
   git clone https://github.com/Anush980/weather-app.git
   cd weather-app
   ```

2. **Get Dependencies**
```bash

```
3. **Run the App**
```bash
flutter run
```
